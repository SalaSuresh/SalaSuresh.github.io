Wait Once

WaitOnce is a seamless appointment booking platform that allows users to schedule, manage, and track appointments with therapists and service providers. It supports waitlisted and recurring bookings, while the backend portal enables providers to manage availability and appointments efficiently. 

France Travail

Pole Emploi (France Travail): Pole Emploi is an application project for HP MFP printers to scan and upload documents to the Pole Emploi server from MFP Printer. This project contains 3 different applications one for citizens in France and another application for Pole Emploi office employees and 3rd application is to diagnose the printer and make sure the printer is ready to use applications and scan documents.
